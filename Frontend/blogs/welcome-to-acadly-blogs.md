# Welcome to Acadly Blogs

The Acadly platform now includes:

- A dedicated **blog listing page**.
- A **blog writer page** with mode switching:
  - Markdown mode
  - Rich text mode
  - LaTeX mode
- Metadata-driven storage using `blogs/metadata.json`.

Inline math also works: $a^2 + b^2 = c^2$.

Display math works too:

$$
\int_0^1 x^2\,dx = \frac{1}{3}
$$

Happy writing ✍️
